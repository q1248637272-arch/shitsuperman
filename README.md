# 大便超人

一个原创 HTML5 横版飞行小游戏。当前项目只维护 Cloudflare 在线版；主角、BOSS、装备、道具和背景资源保存在 `assets/`，玩法代码在本目录内。

## 在线地址

https://shitsuperman.pages.dev/

## 在线版维护

- 后续继续修改根目录的 `index.html`、`styles.css`、`main.js` 和 `assets/`。
- 发布前把这些文件同步到 `cloudflare-pages/`。
- 使用 Wrangler 发布 `cloudflare-pages/` 到 Cloudflare Pages 项目 `shitsuperman`。
- 微信分享版、离线分享包和临时截图已不再维护，避免后续版本分叉。

## 操作

- 空格 / W / 方向上：飞行
- J / 方向右：便便弹攻击
- 手机右半屏长按：飞行
- 大便超人会周期自动攻击，每次进化都会进一步提高攻速
- 手机左半屏点按：高伤害重击，会消耗更多能量
- 大招按钮 / K：消耗 50% 最大能量，释放持续 3 秒的大便能量波，冷却 15 秒
- 开始页“移动方式”按钮：可在按住飞行和带阻尼滑动角色之间切换
- 规则页“画质”按钮：默认自动调节，低端手机可手动切到流畅模式
- 滑动模式：按住右半屏上下拖动角色，角色会带一点惯性跟随，左半屏仍然点按攻击
- 按住飞行模式下基础得分效率为 1.2 倍
- 点按左半屏释放高伤害重击；右半屏用于飞行或滑动移动
- 手机横屏体验更好，横屏下界面会自动压缩状态栏和按钮。
- 收集纸卷、能量瓶、护盾、磁铁并击败机器人获得经验；3、5、7、15、25、35、50 级会强化攻击方式。
- 游戏中会随机出现纸卷雨、马桶突袭、臭气云带和肥皂泡乱流等事件，并有旋转路障、酸液桶等障碍。
- 新增顺风门阵事件：穿过风道门会获得能量、连击、技巧评价和短时间顺风加成。
- 局内新增技巧评价：极限闪避、暴击、弱点命中、击破精英和穿门会提升评价；受击会打断评价。评价越高，得分、伤害和实时战力越高。
- 局内奖励改为透明虚框提示，不会暂停游戏，也不会拦截点击。
- 升级天赋改为玩家三选一；战斗继续进行，倒计时结束会自动选择推荐天赋，也可消耗重骰券换一组。
- 商店新增天赋重骰券、复活心核、臭弹炸弹和顺风羽翼，并配套新增图标素材。
- 推进到一定分数会遇到 BOSS，越后面的 BOSS 越强；BOSS 弹道与出手频率已降低，击败后会获得大量奖励并掉落黄金便便道具。
- BOSS 会根据玩家实时战力做小幅软平衡，低战力更公平，高战力会获得更高挑战和奖励。
- 闯关模式共 60 关；31-60 关使用新地图，40/50/60 关会出现 3 个新 BOSS。
- 首页点击“闯关模式”会进入独立选关界面，可查看目标、奖励和限制后开始本关。
- 无尽模式推进到后段后也会切换到新地图。
- 金币可升级大便超人或购买护盾、磁铁、能量道具；背包道具跨关卡和无尽模式保留。
- 大便超人达到指定等级后，可消耗普通材料和 1 颗特殊进化石进化；特殊进化石来自无尽模式 10万、20万、50万、100万分首次达成奖励。
- 成就页会记录累计纸卷、击破、闪避、通关、Boss、装备等目标，完成后可领取奖励并佩戴称号。
- P / Esc：暂停
- 屏幕按钮：移动端操作

## 文件

- `index.html`：页面入口
- `styles.css`：界面样式
- `main.js`：游戏逻辑
- `assets/`：生成并优化后的美术素材
- `cloudflare-pages/`：Cloudflare Pages 发布目录
- `.wrangler/`：Cloudflare 发布工具的本地配置/缓存
